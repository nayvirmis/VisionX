# Documentation

`VisionX_Legacy_Project_Report.docx` is retained as a historical record of the
original prototype. It describes the pre-2.0 cookie and unofficial-client
implementation and must not be used as current architecture or deployment
documentation.

The root `README.md`, `PRIVACY.md`, current source code, migrations, and API
tests describe VisionX 2.0. The 2.0 rebuild uses official X OAuth verification
and official public-post API access and never reads or stores X session cookies.
