# Changelog

## 2.0.0 - 2026-06-12

- Combined the backend and Chrome Extension into one monorepo.
- Replaced X browser-cookie extraction and unofficial fetching with official X
  OAuth verification and public authored-post retrieval.
- Added rotating refresh tokens, restricted CORS, rate limiting, security
  headers, structured errors, Alembic migrations, and health endpoints.
- Added safe DOM rendering, reliable restore behavior, extension options, and
  centralized authenticated API requests.
- Added Docker Compose, Render configuration, GitHub Actions, tests, privacy
  documentation, and release packaging scripts.
- Removed third-party ownership references and replaced the watermarked icon.
